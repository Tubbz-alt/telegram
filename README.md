# telegram
A SlackBuild for Telegram

Telegram is a cloud-based mobile and desktop messaging app with a focus on
security and speed.


## Usage

下载 官方 Telegram 安装包，到本 git 目录下。地址在 telegram.info 里就有。

```
cd telegram/

source ./telegram.info

wget -c $DOWNLOAD
```

以 root 身份运行安装脚本：

	# ./telegram.SlackBuild
